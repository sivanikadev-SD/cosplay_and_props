# Neon Prop Studio - Premium Cosplay Portfolio Template

A high-end, responsive, and interactive website template for an "Anime & Cosplay Prop Maker". Built with Bootstrap 5, Vanilla JavaScript, and a custom Cyberpunk-inspired design system.

## 🚀 Key Features

- **Dual-Theme Support**: Dark and Light mode with automatic system preference detection.
- **RTL Support**: Built-in compatibility for languages like Arabic and Hebrew.
- **Showcase Gallery**: Masonry layout with interactive commission process and waitlist status.
- **Responsive**: Mobile-first design with optimization for all screen sizes.
- **SEO Ready**: Semantic HTML5, structured meta tags, and high-performance assets.
- **Performance**: Skeleton loaders and optimized image placeholders.

## 📂 File Structure

```text
cosplay-template/

```

## 🛠️ Tech Stack

- **HTML5**: Semantic markup for better SEO and Accessibility.
- **CSS3**: Vanilla CSS with variables for theming.
- **Bootstrap 5**: Grid system and foundational components.
- **JavaScript**: ES6+ modules for interactivity.
- **Icons**: Bootstrap Icons (CDN).

## 📄 Implementation Guidelines

- **Contact Form**: Compatible with Formspree/Netlify by replacing the form action.
- **Maps**: Replace the placeholder in `contact.html` with a Google Maps iframe or API.
- **Theme**: Modify `--primary-neon` and `--secondary-neon` in `style.css` to change the brand colors.
